# Node.js Express API Authentication
A Node.js project demonstrating API authentication methods using Express and Axios.

## 📌 Description
This project demonstrates how to work with external APIs using Node.js, Express, and Axios. It covers different authentication methods including No Authentication, Basic Authentication, API Key, and Bearer Token.

The responses are rendered dynamically using EJS.

---

## 🚀 Features

- Fetch random data from API (No Auth)
- Access protected routes using Basic Authentication
- Retrieve filtered data using API Key
- Secure endpoint access using Bearer Token
- Server-side rendering using EJS

## ⚙️ Installation & Setup

2. Navigate to the project folder


3. Install dependencies

npm install


4. Run the server

node index.js


5. Open in browser  

http://localhost:3000


---

## 🔗 API Endpoints

- `/noAuth` → Fetch random data (No Authentication)
- `/basicAuth` → Access data using Basic Auth
- `/apiKey` → Filter data using API Key
- `/bearerToken` → Access secure data using Bearer Token

---

## ⚡ How It Works

- Express server handles routes
- Axios is used to make API requests
- Different endpoints demonstrate different authentication methods
- Data is passed to EJS and rendered dynamically
